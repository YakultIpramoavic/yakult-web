Danh sách game & scripts

Toàn bộ dữ liệu được lấy từ file assets/data/games.js. Bạn chỉ cần sửa file đó để thêm ~100 game.

Không tìm thấy game phù hợp với từ khoá / bộ lọc hiện tại.

Cách người dùng lấy script

Giải thích nhanh 3 bước, bạn có thể tuỳ chỉnh nội dung.

  1. 1

    Chọn game

    Người dùng cuộn xuống danh sách game, dùng ô tìm kiếm để tìm đúng game.

  2. 2

    Mở link tải

    Bấm nút “Tải script” → chuyển sang Telegram / Drive / web khác của bạn.

  3. 3

    Làm theo hướng dẫn

    Hướng dẫn cài đặt / sử dụng bạn để trong channel hoặc trong file đính kèm.

FAQ

Một số câu hỏi thường gặp. Bạn có thể chỉnh sửa nội dung tuỳ ý.

Website này có chứa file không?

Không bắt buộc. Bạn có thể để tất cả file trên Telegram / Google Drive, website chỉ là nơi tổng hợp link.

Có an toàn cho tài khoản không?

Mọi hình thức mod / script đều có rủi ro. Người dùng cần tự chịu trách nhiệm khi sử dụng.

Làm sao thêm game mới?

Mở file assets/data/games.js và thêm 1 object mới vào mảng GAMES. Thêm bao nhiêu game cũng được.

Deploy ở đâu?

Bạn có thể dùng Vercel để deploy miễn phí. Chỉ cần trỏ project vào folder này là xong.
